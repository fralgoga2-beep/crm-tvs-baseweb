Configuración de sincronización en la nube
🔥 Este CRM usa Firebase Firestore (Google) para sincronizar datos entre todos tus dispositivos en tiempo real — gratis hasta 1GB de datos y 50,000 lecturas/día.
Pasos para conectar:
1
Crea un proyecto Firebase
Ve a console.firebase.google.com → "Añadir proyecto" → nombre: tvs-crm
2
Activa Firestore
Build → Firestore Database → Crear base de datos → modo prueba (o producción) → Siguiente
3
Obtén las credenciales
Configuración del proyecto (⚙️) → Tus apps → Web (</>) → Registrar app → Copiar el objeto firebaseConfig
4
Reglas de seguridad sugeridas
Firestore → Reglas → pega esto y publica:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read, write: if true;
    }
  }
}
Pega tu configuración Firebase:
Error al conectar. Verifica tus credenciales.

Tus credenciales se guardan localmente. No se envían a ningún servidor externo.
